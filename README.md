# default

## Project setup

For this project, You will need the Firebase configurations;

```
apiKey,
authDomain,
projectId,
storageBucket,
messagingSenderId,
appId
```

-In the root, rename `.env.sample` to `.env`

### Install dependencies

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
