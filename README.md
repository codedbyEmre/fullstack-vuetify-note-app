```
Vue 3
Composition Api
Vue Router 4
Vuetify 3
Pinia
Vite
Firebase 9
```

## Project setup

For this project, You will need the Firebase configurations;

```
apiKey,
authDomain,
projectId,
storageBucket,
messagingSenderId,
appId
```

-In the root, rename `.env.sample` to `.env`

### Install dependencies

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
